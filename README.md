
# About
The plan is to create a chess game where two players play a best of 3/5/7 1 minute games and the loser gets an upgrade each time like in the game ROUNDS by Landfall Games.


# Setup
Running the frontend:
    `npm run dev`

Running the backend:
    `cd server`
    `dotnet run`
